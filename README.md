# Fleet

Unified meta-orchestrator wrapping [ruflo](https://github.com/.../ruflo), [superpowers](https://github.com/.../superpowers), and [gsd](https://github.com/.../gsd) behind one MCP, skill bundle, and slash-command surface.

See [docs/SPEC.md](docs/SPEC.md) for the design and [docs/RUNBOOK.md](docs/RUNBOOK.md) for operations.

## Quick start

```bash
uv sync --all-extras
make test
```
